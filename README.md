Документация проекта: система заказов пиццы
Проект представляет собой консольное приложение для управления заказами в пиццерии. Система позволяет работать с данными о пиццах, заказах и клиентах, а также выполнять базовые операции по поиску и фильтрации.
Основные сущности системы

Client — представляет клиента. Содержит уникальный ID, имя, телефон и адрес. К клиенту привязан список его заказов. Поддерживаются операции добавления нового заказа, просмотра всех заказов клиента и поиска заказов по конкретному клиенту.
Courier → Pizza — представляет пиццу из меню. Содержит уникальный ID, название, цену и размер (маленькая, средняя, большая). Пиццы можно фильтровать по размеру.
Order — представляет заказ. Содержит уникальный ID, данные клиента, список выбранных пицц, дату создания заказа и общую сумму. Заказ может включать несколько пицц разных размеров и видов.

Запуск проекта
Запуск проекта осуществляется путем запуска скомпилированной части проекта в консоли либо IDE.

Как использовать приложение
После запуска появится главное меню консольного интерфейса. В нём доступны следующие возможности:

-Просмотр полного меню пицц.
-Фильтрация пицц по размеру (маленькая, средняя, большая).
-Создание нового заказа: выбор или добавление клиента, добавление пицц в заказ.
-Поиск и просмотр всех заказов конкретного клиента.
-Просмотр списка всех созданных заказов.
