<h1>Документация проекта: система заказов пиццы</h1>
<h4>Проект представляет собой консольное приложение для управления заказами в пиццерии. Система позволяет работать с данными о пиццах, заказах и клиентах, а также выполнять базовые операции по поиску и фильтрации.
Основные сущности системы</h4>

<p>Client — представляет клиента. Содержит уникальный ID, имя, телефон и адрес. К клиенту привязан список его заказов. Поддерживаются операции добавления нового заказа, просмотра всех заказов клиента и поиска заказов по конкретному клиенту.</p>
<p>Pizza — представляет пиццу из меню. Содержит уникальный ID, название, цену и размер (маленькая, средняя, большая). Пиццы можно фильтровать по размеру.</p>
<p>Order — представляет заказ. Содержит уникальный ID, данные клиента, список выбранных пицц, дату создания заказа и общую сумму. Заказ может включать несколько пицц разных размеров и видов.</p>

<h2>Запуск проекта</h2>
<p>Запуск проекта осуществляется путем запуска скомпилированной части проекта в консоли либо IDE.</p>.

<h2>Как использовать приложение</h2>
<p>После запуска появится главное меню консольного интерфейса. В нём доступны следующие возможности:</p>
<ul>
<li>Просмотр полного меню пицц.</li>
<li>Фильтрация пицц по размеру (маленькая, средняя, большая).</li>
<li>Создание нового заказа: выбор или добавление клиента, добавление пицц в заказ.</li>
<li>Поиск и просмотр всех заказов конкретного клиента.</li>
<li>Просмотр списка всех созданных заказов.</li>
</ul>
